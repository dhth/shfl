Help
===

Keymaps
---

J                    move item one position below
K                    move item one position above
Enter                move item to the start of the list
j / Down             go down
k | Up               go up
[1-9]                move current item to index in list
g                    go to the start of the list
G                    go to the end of the list
w                    write to file
?                    show/hide help view
Esc / q              go back/exit
